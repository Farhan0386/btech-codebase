Lecture 22: Media Queries

Media queries allow CSS to change based on the screen size.

1. The Concept (Slides 8-10)

Without media queries, a desktop website looks "squished" on a phone. With media queries, we can tell the browser: "If the screen is small, stack the boxes vertically instead of horizontally."

2. The Syntax (Slide 15)

The standard syntax looks like this:

@media (max-width: 600px) {
    /* Styles here ONLY apply if screen is 600px or smaller */
    body {
        background-color: lightyellow;
    }
}

3. Coding Question (Slide 32)

Task: Change background color to lightyellow when width is 600px or less.
