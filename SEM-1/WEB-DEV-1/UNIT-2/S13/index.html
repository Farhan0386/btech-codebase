Lecture 13: Positioning & Basic Layout

By default, elements stack top-to-bottom. Positioning helps us break this rule to create layouts like navbars, popups, and card overlays.

1. Static Positioning (Default)

Rule: Elements appear one after another. top and left have no effect.
Box 1 (Static)
Box 2 (Static)

Notice how Box 2 sits directly below Box 1.

2. Relative Positioning

Rule: Element occupies its original space but is visually shifted.
I am a static reference box.
I am Relative!
(Shifted down 20px, right 30px)
I am another static box.

Notice how the "ghost" space of the relative box is still preserved in the flow.

3. Absolute Positioning

Rule: Element is removed from flow. It positions itself relative to the nearest positioned ancestor (anything not static).
Parent (Relative)
Child (Absolute)
Top: 50px, Left: 50px
