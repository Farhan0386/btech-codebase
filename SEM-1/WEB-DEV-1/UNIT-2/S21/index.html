Lecture 21: CSS Units

"Would you build a house without measuring tools?"

1. Absolute Units (Fixed)

These are fixed to a physical size. They do NOT change when the screen resizes.

  • px: Pixels (Most common for screens).
  • in, cm, mm: Physical units (Best for print, bad for screens).
100px
2in
5cm
50mm
200pt
15pc

2. Relative Units (Flexible)

These adapt based on other factors. Essential for Responsive Design.

EM vs REM (Font Relative)

I am the Parent Container (font-size: 20px)

10em
(10 * Parent 20px = 200px)
10rem
(10 * Root 16px = 160px)

Viewport Units (Screen Relative)

50vw (Half Screen Width)
20vh Height
